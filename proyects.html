Starbucks Clone Website

This project is a responsive website created with HTML and CSS, replicating the official Starbucks homepage design. It focuses on recreating the layout, typography, and color scheme to match the original site. Through this project, key web development concepts like Flexbox, Grid, and responsive design techniques were applied to ensure compatibility across various devices and screen sizes. It serves as a showcase of front-end development skills and attention to detail.

Rick And Morty API

This project uses the Rick and Morty API to explore and display information about the characters in the series. Implemented with HTML, CSS and JavaScript, it allows you to search for characters, view details such as their origin, species and status, and filter results by various categories. This project was designed as a hands-on exercise to learn how to consume APIs, handle dynamic data, and work with JSON responses in an interactive web environment.
